Game development for the web

Tatsu is a game engine, using the latest in web technologies to provide a simple programming model for game development. This project is very much still a work in progress. Stay tuned to this site for details.

Goal

This project started out as a way to learn how to write WebGL code, and as a way to write reusable components for demos and games. I hope to refine the project to make it a full-fledged game toolkit for the web.

Development

Currently, this project is in its infancy. I would love to hear your feedback and suggestions on the direction of Tatsu so far.

The first milestone for this project is to build something useable for the 10th anniversary of the Ludem Dare online game jam.
